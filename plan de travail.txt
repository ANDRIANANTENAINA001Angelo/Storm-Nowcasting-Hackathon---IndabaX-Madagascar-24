Plan de travail suggéré :

1- Analyse des données :

Explore les variables fournies (temps, localisation, intensité, etc.).
Nettoie les données si nécessaire (valeurs manquantes, anomalies).

2- Compréhension du phénomène réel : 

Comprendre les variables et leur liaison dans la formation des tempêtes
 

3- Choix du modèle :

Approche simple : Modèle de régression logistique pour obtenir des probabilités.
Approche avancée : Modèles complexes comme les réseaux neuronaux, les forêts aléatoires ou les XGBoost.


4- Optimisation :

Utilise l’AUC-ROC comme métrique principale pour chaque horizon (1h, 3h).
Ajuste les hyperparamètres et compare différentes architectures.


5- Validation croisée :

Teste ton modèle avec des données distinctes pour éviter le surapprentissage.


6- Préparation du fichier CSV :

Assure-toi que les colonnes et le format respectent les spécifications.


7- Description méthodologique :

Décris brièvement (quelques phrases) le type de modèle utilisé, pourquoi tu l’as choisi, et comment il fonctionne.
